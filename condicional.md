# Condicional 
Condicional pode ser entendido como uma escolha. Para fazer essas escolhas usamos operadores lógicos ou matemáticos. Exemplo.:
- +,-,*,/
- <,>,==
- and, or, not
As palavras reservadas para isso são: if, else, else if
## _IF_
A condição if significa se. Se atender as condições faça uma coisa
## _ELSE_
A condição else significa senao, quer dizer que, a primeira condiçã não foi aceita, então estamos no else.
## _ELSE IF_ 
A condição else if significa senao se. Essa condição não pode ser nem a primeira nem como a últims condição pois representa como se fosse uma condição intermediaria
# Operador Ternário
Essa operação é bem parecida com _IF ELSE_, contudo usaremos os operadores (*?* 'IF' e *:* 'ELSE')
